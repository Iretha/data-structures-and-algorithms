# Data Structures vs APIs

APIs are build on top on the Data Structures and they add additional functionalities to the basic data structures.

Data Structure| |APIs
---|---|---
Array|->|ArrayList
Linked List|->|Stacks & Queues
Binary Tree| -> | Binary Heap
